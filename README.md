# AI Real Estate Deal Intelligence Machine

## Phase 0

This repository now contains the initial Phase 0 runtime skeleton for the AI-powered real estate deal intelligence machine.

### What is included

- A mission-control dashboard object with a machine summary.
- A mock provider registry that clearly labels providers as mock integrations.
- Test coverage proving the dashboard shape and provider labeling.

### Important note

All Phase 0 providers are mock providers. They are intentionally labeled so the system does not misrepresent mock data as a live production integration.
