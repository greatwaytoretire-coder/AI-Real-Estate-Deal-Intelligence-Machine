# PHASE 6 — AUTONOMOUS UNDERWRITING INTELLIGENCE

## Goal

Establish the initial underwriting-analysis foundation for comparable sales, ARV, and repair estimation.

## Scope

This phase introduces a bounded local foundation for:

- identifying comparables,
- ranking comparable properties,
- estimating ARV,
- detecting outliers,
- estimating repair scope and cost,
- assigning confidence,
- providing assumptions and source provenance.

## Covered property components

- roof
- HVAC
- electrical
- plumbing
- foundation
- kitchen
- bathrooms
- flooring
- windows
- exterior

## Required estimate contract

Every estimate in this phase must show:

- range
- confidence
- data source
- assumptions

## Important distinction

The system must visibly distinguish estimated values from verified data. This phase provides the model contract to do that clearly at the foundation layer.

## Constraints

- No downstream acquisition or disposition workflow is added in this phase.
- Local mock evidence is used for the underwriting foundation.
- No live production valuation inputs are introduced.
